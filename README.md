# gis-service

### GET /tiles/{map_style}/{z}/{x}/{y}
**map_style** -> Имя папки в **root_dir/data/tiles**
**z** -> зум (1 - 17) int
**x** -> latitude int
**y** -> longitude int 

### GET /height/{lat}/{lon}
**lat** -> latitude float64
**lon** -> longitude float64
Возвращает высоту int
